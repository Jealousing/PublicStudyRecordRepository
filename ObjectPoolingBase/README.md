# Unity-ObjectPooling

## 개요
 게임에서 많은 오브젝트를 생성하고 파괴하는 과정에는 많은 비용을 소모합니다.
 비용을 최소화를 위한 방법중 하나인 오브젝트풀링을 사용할수 있습니다.
 
 유니티 2021버전 이후 추가된 IOobjectPool 기능을 사용하고,
 Dictionary와 Tuple을 활용한 예시입니다.
