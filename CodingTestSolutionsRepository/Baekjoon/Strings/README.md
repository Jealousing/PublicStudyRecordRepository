### Key Takeaways  
아스키코드를 이용한 문자열 문제와 문자열 자체를 다루는 문제들로 구성  
getline(cin, string); -> 줄바꿈(개행문자)까지 입력을 받기위한 함수  
string.substr(n) -> n+1 이후 추출 / n,m n부터 n+m까지 추출  
reverse(string.begin(),string.end()) -> 뒤집기  
stoi(string) -> int로 변환
