
적(몬스터) 구상

필요 스크립트
1) EnemyInfo : 적의 기본적인 정보 관리하는 스크립트 

2) EnemyStateSystem : 적의 상태에 따른 행동 패턴 실행 스크립트
	2-1) EnemySearch : 타겟을 찾기 전의 상태
	2-2) EnemyTracking : 타겟을 찾아 타겟의 방향으로 이동하고 그에 맞는 행동을 하도록 하는 상태
	2-3) EnemyAttack : 공격패턴을 관리하는 시스템
					1. 체력에 따른 패턴 ( 들어온 순서대로 패턴 출력 )
					2. 랜덤 행동 패턴 (스킬과 같으며 쿨이아닌 패턴 목록중 랜덤 실행)
					3. 일반 공격

3) RecognitionSystem : 타겟관리 시스템
